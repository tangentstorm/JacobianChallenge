\begin{lemma}[Lip-cycle and Trace preservation]
\label{lem:impl-trace-lip}
\lean{JacobianChallenge.TraceDegree.pullbackTraceLift_preserves_lattice_raw, JacobianChallenge.TraceDegree.cycleLipChain_obligation, JacobianChallenge.TraceDegree.cycleLipPath_obligation, JacobianChallenge.TraceDegree.PiecewiseC1PathRegularity, JacobianChallenge.TraceDegree.instPiecewiseC1PathRegularity}
\leanok
The trace lift preserves the period lattice. The two named \(C^1\)-cycle obligations
(\texttt{cycleLipPath\_obligation}, \texttt{cycleLipChain\_obligation}) reduce a uniform Lipschitz
bound on chart-lifted paths to a uniform piecewise-\(C^1\) hypothesis on the
representative paths, via the mean-value inequality.

The uniform piecewise-\(C^1\) hypothesis itself is isolated as a typeclass
\texttt{PiecewiseC1PathRegularity X} on the surface. Production code in
\texttt{Jacobian/TraceDegree/PushforwardBasis.lean} and \texttt{PullbackBasis.lean} carries
this typeclass as an instance hypothesis rather than claiming to prove the false unconditional
form (a nowhere-differentiable continuous \texttt{Path a b} would be a counterexample).
The single named sorry-instance lives in
\texttt{Jacobian/TraceDegree/PiecewiseC1Instance.lean} and is the auditable boundary at which
the assumption enters. Honest discharge requires either enriching \texttt{IntegralOneCycle X}
to carry smoothness data, or restricting the period pairing's cycle domain to a piecewise-smooth
subclass shown to span $H_1(X,\mathbb{Z})$.
\end{lemma}
